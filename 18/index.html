259関数内で使う定数や変数の影響範囲（スコープ）について知る

構文 意味
{} ブロックスコープ
  • 変数や定数は{}で囲まれたブロックにて使用できる
  • 変数・定数が有効な範囲を「スコープ」、ブロックのスコープを「ブロックスコープ」という
  • すべてのスコープの外側（トップレベル）は「グローバルスコープ」となり、すべてのブロック内で使用可能
  • 「var」はブロックスコープに対応できないため、定数・変数のスコープがわかりづらくなりバグの原因になる

ブロックの外側だとエラー

{
	{
		const a = 20;
	}
	//Error
	console.log(a);
}

グローバルスコープ

const a = 20;
{
	//OK
	console.log(a);
}

var宣言（非推奨）

{
	{
		var a = 20;
	}
	//Errorにならない
	console.log(a);
}
